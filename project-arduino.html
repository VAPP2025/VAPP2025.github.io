Overview

This system uses an Arduino Uno to continuously poll environmental sensors and output structured data over serial. Alerts trigger when readings exceed defined thresholds — activating an LED buzzer module. Add your full description here.

What I Built

  • Sensor polling loop reading DHT22 (temperature/humidity) and PIR (motion)
  • Serial data output formatted for easy logging and parsing
  • Threshold-based alert system with LED + buzzer
  • Modular code structure allowing easy sensor swaps

What I Learned

  • Working with real sensor datasheets and timing constraints
  • Debugging hardware issues using serial monitor
  • Power management considerations for microcontrollers
  • Writing clean, readable embedded C code
